 & 
	\multicolumn{2}{L{\columnT}}{
	\vspace{-5mm}
	\small
	\begin{itemize}[leftmargin=*, parsep=0pt, topsep=0pt, itemsep=0pt]
				\item True pay as you go pricing
				\item DevOps application management solution supporting the entire application lifecycle -- resource provisioning, configuration management, application deployment, software updates, monitoring, and access control
				\item Three basic concepts:
					\begin{itemize}[leftmargin=*, parsep=0pt, topsep=0pt, itemsep=0pt]
						\item Stack: Set of Amazon EC2 instances, including one or more layers
						\item Layer: Description (Chef recipe) how to configure an instance. \ac{AWS} provides built-in layers -- for instance for Rails, Ruby, Node.js, HAProxy, and Ganglia. However, also custom layers can be built. Furthermore, layers can be configured to scale-up as well as scale-down automatically, both, load- and time-based scaling is possible.
						\item Application: \ac{AWS} OpsWorks receives the application from a repository (i.e. Git, SVN, HTTP, or Amazon S3) and deploys the source code as configured.
					\end{itemize}
				\item \ac{AWS} OpsWorks can be used via the \ac{AWS} Management Console, \ac{SDK}, or command line interface\vspace{-\baselineskip} 
		\end{itemize}
	} \\ \midrule
